## 附：LLM 使用说明

实现任何 UI 时，必须遵守：
1. 颜色只能使用第 2 节定义的 token，不得直接写色值
2. 字体只能使用第 3 节定义的规格
3. 间距只能使用第 4 节定义的 token
4. 有特殊要求的组件，先查第 6 节，再实现
5. 模块特有的 UI 要求，以功能文件的第 5 节为准，本文件为基础默认值

### 遇到问题时
```
普通失败 → 修正 → 重试
连续失败3次 → 元反思
  → 我在解决什么问题？
  → 我的方向是否本身就错了？
  → 我是否在重复同一个错误？
  → 换完全不同的思路重新开始
元反思后再失败2次 → 上报人工介入
  说明：任务描述 / 已尝试方案 / 卡住原因
```
